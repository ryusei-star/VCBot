## 🎧 VCBot

Spotify、SoundCloud、YouTube、ニコニコ動画、Twitch、Pornhub に対応した、高機能な音楽Botです。  

[![MIT License](https://img.shields.io/github/license/ryusei-star/VCBot?color=44CC11&style=flat-square)](LICENSE)

### 対応機能
- ⏯️ 再生・一時停止・スキップ
- 🔁 ループ・キュー管理
- 🔍 複数プラットフォームからの検索再生
- 📁 ファイル再生にも対応
- 🎤 聞き専読み上げにも対応
